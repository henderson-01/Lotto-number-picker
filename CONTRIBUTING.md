# Contributing to EuroMillions Number Picker 🎲

First of all, thank you for showing interest in this project!

## The Learning Journey

**EuroMillions Number Picker** is a personal learning project. It was built to practice specific software development concepts, including GUI development with `CustomTkinter`, random number generation, and improving UX/UI design skills in Python.

Because this repository serves as a snapshot of my individual learning journey and a portfolio piece, **Pull Requests (PRs) will not be merged into the main branch.**

## How You Can Participate

Even though I am not merging external contributions, you are more than welcome to:

- **Fork the Repository:** If you like the logic or the UI, please feel free to fork the project and use it as a base for your own experiments or improvements.
- **Open Issues:** If you find a bug or have a suggestion, feel free to open an issue. While I may not implement every change, I value the feedback as part of my learning process.
- **Discussions:** Feel free to reach out with questions about how a specific part of the code works!
